対応言語一覧

Antigravityは主要なプログラミング言語から最新のフレームワークまで、幅広くサポートしています。

フロントエンド

  • HTML5 / CSS3 (Sass, Less)
  • JavaScript (ES6+)
  • TypeScript (推奨)
  • React / Next.js
  • Vue.js / Nuxt.js
  • Tailwind CSS

バックエンド

  • Node.js (Express, NestJS)
  • Python (Django, FastAPI)
  • Go
  • Ruby (on Rails)
  • PHP (Laravel)
  • Java (Spring Boot)

インフラ・ツール

  • Docker / Docker Compose
  • Terraform
  • GitHub Actions / GitLab CI
  • SQL (PostgreSQL, MySQL)
  • NoSQL (MongoDB, Redis)

モバイル・その他

  • Flutter (Dart)
  • React Native
  • Swift / Kotlin
  • Markdown (ドキュメント)
  • Shell Script (Bash/Zsh)

未記載の言語について

Antigravityは高度な推論能力を備えているため、上記に記載のないマイナーな言語や、独自のDSL（ドメイン固有言語）についても、コードのパターンを読み取ることで対応可能な場合があります。

特定のライブラリやフレームワークの知識をAIに強化したい場合は、AGENTS.md にそのドキュメントのURLや要約を記述することを推奨します。
