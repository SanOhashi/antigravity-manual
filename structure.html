構造説明

AIが情報を解析する際の「論理的階層」に基づいた配置を解説します。

解析ロジックに基づいた配置

MyProject/
├── AGENTS.md              // なぜルート：全ファイル解析の最上位制約
├── instructions.md        // なぜルート：現在のタスクをAIに直視させるため
└── .env                   // なぜ重要：AIの推論から物理的に遮断すべき領域
Analysis Logic

設定ファイルをルートに配置することで、AIはファイルをスキャンする際、常に「このプロジェクトの憲法（AGENTS.md）」と「現在の目的（instructions.md）」を最優先のフィルターとして適用できます。
